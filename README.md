# 🚧 RoadAlert - Municipal Road Damage Portal

A web-based citizen portal for reporting road damage (potholes, cracks, etc.) to municipal authorities. Built with HTML, CSS, JavaScript, and Firebase.

## ✨ Features
- **User Authentication** - Login/Register with Firebase Auth
- **Damage Reporting** - Submit detailed road damage reports with location
- **Photo Upload** - Attach up to 5 images per report (Cloudinary integration)
- **Severity Levels** - Classify damage as Low, Medium, or High priority
- **Ticket Generation** - Get unique complaint IDs (e.g., PMC-2024-XXXXXX)
- **Responsive Design** - Works on mobile and desktop

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Authentication:** Firebase Auth
- **Database:** Firebase Realtime Database
- **Storage:** Cloudinary (for image hosting)
- **Hosting:** GitHub Pages

## 📱 How to Use
1. Visit the live link above
2. Create an account or continue as guest
3. Fill in damage details and location
4. Upload photos of the damage
5. Submit and receive your ticket ID
6. Track status (future feature)

## 🚀 Setup (For Developers)
If you want to run this locally:
1. Clone this repository
2. Replace Firebase config with your own keys
3. Replace Cloudinary preset with your own
4. Open `index.html` in a browser

## 🔒 Security Note
- Firebase API keys are public (safe for web apps) but protected by Firebase Security Rules
- Images are uploaded to Cloudinary with unsigned presets
